- 新仓是 `Electron + electron-vite + Vue 3 + TypeScript + Naive UI` 重构版，不是旧项目逐文件搬运。
- 主架构是 `Renderer -> Preload -> Main Process Service -> 本地 JSON / 本机 SDK / 网络请求`。
- 已经完成现代工程骨架、本地资料模块、HTTP 集合、AI SDK Bridge、统一备份恢复、旧数据导入、核心外观配置和一批纯前端工具。
- 仍未完成旧项目 50+ 页面的一比一独立页面迁移；复杂工具多数还只是未迁移或轻量替代。
