## 1. 常用入口

### 本地开发

```bash
pnpm install
pnpm run dev
```

### 最小检查

```bash
pnpm run test
pnpm run typecheck
pnpm run lint
pnpm run build
```

### AI 本机链路 smoke test

```bash
pnpm run smoke:ai
```

这些命令的适用场景、边界和 CI 打包说明，请看 `01-开发、打包、测试与目录结构.md`。

## 3. 当前统一口径

- 当前开发基线以 `package.json` 脚本为准。
- AI 功能走本机 Codex / Claude SDK Bridge，不走旧项目手写 HTTPS Provider。
- 外观设置当前是 Renderer 顶部外观条即时生效并写入 `localStorage`，不是旧项目设置弹窗的一比一复刻。
- 继续开发时先看主线文档，不再在本文件维护重复的页面说明。
