# Doggy Toolbox UI 重构终极准则 (Zen-iOS Hybrid 像素对齐版)

> **核心使命：** 抹去“网页感”，注入“精密感”。将工具箱打造成如瑞士手表般紧凑、如 iOS 系统般丝滑的专业生产力工具。
> **参考母版：** 根目录 `zen-prototype/` (所有视觉冲突以此为准)。

---

## 0. 全局物理基因 (DNA Rules)

### 0.1 字体与比例 (The 12px Standard)
- **基础字号**：全站主文字强制 **12px** (部分场景可为 13px)，严禁出现 14px。
- **Eyebrow 标签**：强制 **10px**，`font-weight: 800`，字间距 `0.15em`，全大写。
- **行高 (Line Height)**：统一为 `1.5`，确保紧凑但不粘连。

### 0.2 间距与 Padding (The 30% Compression)
- **卡片内边距**：从 24px 压缩至 **16px** (核心工作区) 或 **12px** (辅助区)。
- **全局 Gap**：所有 `flex/grid gap` 统一为 **12px** 或 **16px**，禁止出现大空洞。
- **容器圆角**：大容器 **28px**，功能卡片 **14px**，按钮/输入框 **10px**。

### 0.3 物理材质 (The Glassmorphism)
- **材质**：`rgba(255,255,255,0.7)` + `backdrop-filter: blur(40px)`。
- **双层描边 (核心)**：
  - **外层**：`box-shadow: 0 12px 24px -8px rgba(0,0,0,0.06)`。
  - **内层**：`inset 0 0 0 1px rgba(255,255,255,0.8)` (暗色模式切为 `white/0.05`)。

---

## 1. 页面级重构指标

### Stage 1. HTTP 实验室 (精密工作台)
- **布局**：严格三栏。左侧 Sub-Sidebar (220px) | 中间配置 (Flexible) | 右侧响应 (Flexible)。
- **URL 输入栏**：高度压至 **36px**，背景色 `#F2F2F7`。
- **KV 编辑器**：每一行（Key-Value 对）高度 **28px**。移除 NInput 边框，改为 1px 细底边。
- **响应区 (Response)**：必须使用 `shadow-inner` 营造下凹质感，代码字号 **11px**，背景冷灰。
- **方法选择器**：采用 Pill (半圆) 形状，字号 **10px**，颜色需高亮区分 (GET-绿, POST-蓝)。

### Stage 2. AI Chat (灵动岛模式)
- **气泡**：内边距强制 **10px 14px**。AI 气泡背景纯白，用户气泡 iOS 系统蓝 (`#007AFF`)。
- **非对称圆角**：AI 气泡左下圆角 4px；用户气泡右下圆角 4px。其余角均为 24px。
- **输入岛**：悬浮在内容区底端，宽度 **80% - 85%**，高度 **52px**。
- **模型条**：顶部 `ai-toolbar` 高度 **48px**，集成极致毛玻璃模型切换器和物理滑动条。

### Stage 3. 开发工具 (紧密矩阵)
- **卡片体积**：每个 `zen-card` 面积缩小 30%。
- **图标盒**：`icon-box` 尺寸 **36x36px**，背景使用内阴影。图标字号 **18px**。
- **排序交互**：实现拖拽排序，拖拽瞬间卡片 `scale(1.05)`，阴影深度加倍。

### Stage 4. 命令管理 (终端沉浸)
- **列表项**：单行高度压缩至 **40px**。
- **预览区**：采用沉浸式黑底绿字，高度固定，移除标题栏，直接通过 `eyebrow` 标识。
- **执行按钮**：放在卡片右上角，小号圆钮风格，直径 **28px**。

### Stage 5. 凭证管理 (保险库)
- **列表密度**：单行高度 **48px**。
- **敏感掩码**：星号 `*` 采用等宽字体，间距加宽，增强专业感。
- **侧栏详情**：取消独立 Card 嵌套，直接在页面 Canvas 上通过 1px 细线分隔。

## 2. 交互与动效规范 (Haptics)
- **点击回弹**：所有 `button`, `.nav-item`, `.zen-card` 必须绑定 `active:scale-95`。
- **过渡速度**：全站统一使用 `0.4s cubic-bezier(0.4, 0, 0.2, 1)`。
- **入场位移**：页面切换时，内容从底部 **10px** 处淡入，禁止生硬切换。
