- repository 层保留清晰边界，后续如果凭证、HTTP 历史、AI 会话规模增长，再迁移到 SQLite 或其他本地数据库。

- 凭证管理已支持新增、编辑、删除、搜索、显示/隐藏、复制账号和复制密码；旧项目批量导入与拖拽排序后续并入“旧数据导入助手 / 备份恢复”。

- 节点列表已补齐 Electron main / preload / IPC 合约，Main Process 通过 `appData/storage/nodes.json` 管理本地节点库；当前先覆盖节点资料管理最小闭环，不在这一轮引入订阅拉取、转换或复杂协议解析。
- 新仓统一备份当前覆盖 `commands + credentials + prompts + nodes + httpCollections + aiSettings`，恢复时按所选模块覆盖写回；节点、HTTP 集合和 AI 设置会按备份里的模块数据整体恢复。
- 旧数据导入助手已支持识别旧项目总备份 JSON 和旧 Prompt 模板导出 JSON；前者可按模块覆盖迁移命令/凭证/节点，后者按分类名称与模板标题做合并导入。
